Pregunta frecuente

¿Sirve para NAS y discos de red?

  • Sí. Si el share del NAS está montado (SMB/AFP/NFS), se escanea igual que un disco local. El modelo de datos ya contempla el tipo nas.
  • Por agregar para que sea 1-clic: botón "Escanear carpeta…" e identificar el NAS por su URL de red.
  • Matices honestos: el escaneo por red es más lento (latencia por archivo) y la huella del volumen funciona distinto que en discos locales.

Bajo el capó

Cómo está hecho

  • Tauri 2 + Rust para el motor: binario chico, rápido, multiplataforma. El trabajo pesado nunca traba la UI.
  • React + TypeScript para una UI moderna y accesible (modo oscuro, atajos, virtualización).
  • SQLite + full-text search: escala a millones de archivos en un catálogo portable de un solo archivo.
  • Seguridad por diseño para el conector: solo lectura, autenticado por dispositivo, cifrado, sin exponer nada fuera del catálogo.
